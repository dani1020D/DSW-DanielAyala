    • Proposito de sitio web

    El proposito de esta página es profundizar sobre las tematicas flexbox y Listas y enlaces. Y la importancia que tienen las Hojas de estilo de cascada que nos permiten visualizar la pagina web de una manera atractiva y llamativas para el cliente, en este caso la utilizaremos para fines educativos.

    Introducción

    El flexbox se utilizó debido a problemas que se presentaban en las versiones anteriores de hoja de estilos de cascada ya que código era mucho más extenso. Este se creó para solucionar una parte larga y tediosa que había que hacer, haciendo que estos contenedores fueran más organizados dentro de su contenedor para que la página web tuviera un orden que le facilitaría la programación del lenguaje HTML5 al programador. También se habla de listas y enlaces las cuales se utilizaran en el caso lista según su caso, Ya sea lista no ordenadas (Estas se utilizan para materiales requeridos para algo específico) o listas ordenadas que nos indica un procedimiento indicando el paso a paso.

    Lista de Definiciónes

    Como su nombre lo indica es la definición de cada una de las palabras como una especie de diccionario.

    Enlaces

    Los enlaces son herramientas que se utilizan para enlazar, esto también se le conoce como hipervínculo, que nos permite ampliar la información sobre un tema o una investigación que nos permite ampliarla por estar relacionada con el tema.
